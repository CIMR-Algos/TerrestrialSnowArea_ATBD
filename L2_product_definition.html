Level-2 Product Definition#

The content of the Level-2 TSA product files is defined in Tables 1, 2, 3 and 4. The product is provided in netCDF format and contains the variables given in Table 1.

Table 1 NetCDF Group: Processed Data (tbc).#

n/a

(nx,ny)

Note

The CIMR Level-2 TSA product is a gridded product. The dimensions of each variable in the Level-2 file refer to the (nx,ny) dimensions of the product grid i.e. EASE-Grid 2.0 polar projection.

Each data variable of the processed data in Table 1 holds conventional attributes following NetCDF Climate and Forecast (CF) Metadata Conventions Version 1.7 (CF-1.7) or above as given in Table 2.

Table 2 Standard variable attributes (tbc).#

Name

Description

standard_name

A standard name that references a description of a variable’s content

long_name

A descriptive name that indicates a variable’s content

fill_value

Value that will appear in the dataset when the data is either missing or undefined

units

Unit of measure

Table 3 Status flag (tbc).#
